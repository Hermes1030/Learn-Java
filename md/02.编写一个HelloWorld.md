# 开发流程

1. 编写程序

- 新建文本文档，修改名称为 HelloWorld.java
- 写入代码

```java
public class HelloWorld {
    public static void main(String[]args) {
        System.out.println("Hello World");
    }
}
```

1. 编译程序

```java
 javac HelloWorld.java
```

1. 运行程序

 ```java
 java HelloWorld
 ```

- 执行结果

> Hello World.
