### 开发流程

- 编写程序

 1. 新建文本文档，修改名称为 HelloWorld.java
 2. 写入代码

  ```java
  public class HelloWorld {
      public static void main(String[] args) {
          System.out.println("Hello World.");
      }
  }
  ```

- 编译程序

```java
 javac HelloWorld.java
```

- 运行程序

 ```java
 java HelloWorld
 ```

- 执行结果

> Hello World.
