### 数据输入

#### 1）操作步骤

1. 导包

```java
import java.util.Scanner;
```

2. 创建对象

```java
Scanner info = new Scanner(System.in);
```

3. 接收数据

```java
int info_recive = info.nextInt();
```

> info 属于变量，可自定义。
