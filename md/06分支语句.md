### 01 流程控制语句
#### 顺序结构
    按照代码的先后顺序依次执行。
#### 分支结构
##### 1）if 语句
```java
if (关系表达式){
    语句体;
}
```
执行流程：
1. 首先计算表达式的值
2. 如果关系表达式的值为true就执行语句体
3. 如果关系表达式的值为false就不执行语句体
4. 继续执行后面的语句内容

##### 2）if...else...语句
```java
if(关系表达式){
    语句体1;
}
else{
    语句体2：
}
```
执行流程：
1. 首先计算关系表达式的值
2. 如果关系表达式的值为true就执行语句体1
3. 如果关系表达式的值为false就执行语句体2
4. 继续执行后面的语句体内容

##### 3）elseif语句
```java
if(关系表达式1){
    语句体1;
}
else if(关系表达式2){
    语句体2;
}
・・・
else{
    语句体n+1;
}
```
执行流程：
1. 首先计算关系表达式1的值
2. 如果为true，就执行语句体1，如果为false就计算关机表示2的值
3. ・・・
4. 如果没有任何关系表示为true，就执行语句体n+1
